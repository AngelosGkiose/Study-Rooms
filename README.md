
Study Rooms
===

---

Clone the repository:

```shell
git clone https://github.com/AngelosGkiose/Study-Rooms.git
```

---

Run:
Για να τρεξετε την εφαρμογη θα πρεπει να τρεχει παραλληλα η εξωτερικη υπηρεσια HUA NOC.Επισης η εφαρμογη ειναι σε java 17.  
Πριν τρέξετε την εφαρμογή, είναι απαραίτητο να εκκινήσετε τη βάση δεδομένων (PostgreSQL) σε Docker container. Απαιτείται να έχετε εγκατεστημένο το Docker Desktop.
Με την εντολη  docker-compose up -d
```shell
cd studyrooms 
./mvnw spring-boot:run  # MacOS / Linux
./mvnw.cmd spring-boot:run  # Windows
```

---

Open in browser: [localhost:8080](http://localhost:8080)
